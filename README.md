# 💰 Expense Tracker

A modern Expense Tracker web application that helps users manage their income and expenses efficiently. Users can add, edit, and delete transactions while tracking their overall balance, income, and expenses through an intuitive and responsive interface. All data is stored locally to ensure persistence across browser sessions.

## ✨ Features

- Add income and expense transactions
- Edit existing transactions
- Delete transactions
- View total balance
- Separate income and expense summaries
- Transaction history
- Responsive modern UI
- LocalStorage persistence
- Instant balance updates
- User-friendly interface

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/subhamswarupa/Expense-Tracker.git
```

Navigate into the project

```bash
cd Expense-Tracker
```

Open `index.html` in your preferred web browser.

---

## 📖 How to Use

1. Enter the transaction title.
2. Enter the amount.
3. Select whether it is Income or Expense.
4. Click **Add Transaction**.
5. View updated balance and transaction history.
6. Edit or delete transactions as needed.
7. Refresh the page—your data remains saved automatically.
